This application requires openh264.dll, version 2.3.1 32-bit.

Download http://ciscobinary.openh264.org/openh264-2.3.1-win32.dll.bz2 and extract it (using e.g. 7-zip).
Rename openh264-2.3.1-win32.dll and put it next to application executable.
